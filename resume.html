resume hari ini

hari ini kita belajar tentang git, berikut adalah command-command dari git.

  • git init = untuk initialized git
  • git add . = untuk save semua file
  • git commit -m = untuk menambahkan commit dalam github
  • git branch -m = untuk membuat branch
  • git fetch = untuk mengetahui kode yg terupdate
  • git remote add origin = untuk menambahkan link github kita
  • git push = untuk mengupload hasil ke github
  • git clone = clone repository orang
  • git checkout = untuk keluar dari branch
  • git remote show origin = untuk melihat filenya ada dimana
